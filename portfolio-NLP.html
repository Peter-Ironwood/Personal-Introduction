PDF Question Answering

Natural Language Processing

The objective of this project is to generate reasoning question and answer pairs based on the attributes and relationships of PDF objects. There are different question types: counting, comparison, relations and content. There are two VQA models used for PDF QA performance testing: MAC (Memory, Attention, Composition) and LayoutLM.

Before training the models, the question-answer pairs need to be balanced. The unbalanced question-answer pairs could cause inadequate training on certain question types. I used data augmentation to increase the number of the question-answer pairs before I balanced the question-answer types.

The final delivery is a web page that the user could type in the question and page number. The PDF pages are listed on the screen with corresponding numbers. I used python-flask to build the page and connected it to the trained model.

There are two models to choose from. By default, the model is the MAC model, users could choose to use the MAC model or MAC + LayoutLM.

Linear-regression

Linear Regression is a linear modeling method which uses a linear function that could represent relationships between a group of data points. The linear regression focuses on the conditional probability distribution. In supervised machine learning, linear regression is one of the fundamental algorithms. Exponential-regression

Exponential Regression is similar to linear regression but it represents a change which is not at a fixed rate. The growth of population, temperature or radioactive decay can be modeled by exponential functions. Clustering-progress

Clustering aims to divide small different groups in a big group. It is widely used in pattern recognition, image analysis, and machine learning. One of the commonly used algorithms is k-means. The basic concept of k-means is to specify the number of clusters: k and find the nearest neighbour of each center.

What I could do

Machine Learning

In the project of machine learning, I used a transition matrix to solve problems with noise label classification. The object is to build at least two different noise robustness classifiers on random label noise classification and build a transition matrix estimator to estimate transition matrix. I was responsible for building two classifiers and generating transition matrices. The prediction accuracy is 50% without transition matrix and 80% with transition matrix, respectively.

Natural Language Processing

In the project of natural language learning, I used algorithm on PDF question and answering. The object is to use CLEVR algorithm to generate question answer pairs on PDF files and use two different neural networks to train and test with generated question answer pairs. I was responsible to modify the given PDF and generate question answer pairs. In order to increase the training-testing result, I tried to even the number of different types of question. The final delivery is a web interface that I created with python-flask that connects to the trained model.

Data Visualization

In the project of data visualization, I used either Tableau or Power BI to create KPIs in the dashboards. The object is to create dashboards that indicate the effectiveness of different methods of COVID-19 spread control. I retrieve data from the NSW Health web site and clean the data before I use it. The result indicates that vaccination and border control have better effectiveness on controlling COVID-19 spreading.
